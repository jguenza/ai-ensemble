The Project

AI Ensemble is an academic experiment exploring whether a coherent, structured book can emerge from repeated interactions with an AI agent that starts from a clean initial state in every session — without memory, without context accumulation, and without long-term reinforcement.

The hypothesis is that structure emerges from ensemble behaviour, similar to how statistical mechanics derives macroscopic order from microstate distributions.

How the Book Will Emerge

All interactions with the agent are logged. A second system will later read these logs, extract conceptual structures, and assemble them into an evolving book.

Can knowledge emerge from an ensemble of conversations with a memoryless agent?

The long-term vision is a self-building academic manuscript grounded entirely in human–AI dialogue.
