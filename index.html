About the Project

AI Ensemble is an academic experiment exploring whether a book can emerge from the continuous interaction between a user and a clean-state AI agent.

Every session begins from first principles. Nothing is remembered between conversations. All meaning, structure, and theory must re-emerge from scratch each time — like deriving statistical mechanics from microstates with no priors.

All conversations are logged privately. A second system (not public yet) processes these logs to build a dynamic and evolving “living book” about:

  • Statistical mechanics
  • Epistemology and uncertainty
  • Entropy, information and inference
  • Ockham’s razor as a guiding principle
  • The emergence of structure from clean initial states

Project Vision

The goal is not to write a book, but to discover whether a book can be grown from fundamental interactions — the same way thermodynamic laws emerge from the combinatorics of microstates.

If statistical mechanics teaches us that order emerges from ensemble behaviour, this project asks:

Can knowledge emerge from an ensemble of conversations with an amnesic agent?
